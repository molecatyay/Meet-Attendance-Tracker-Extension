Settings

Track by name list

When you record attendance, the current attendees will be compared with the given name list, and generate a list of absentees.

Best for meetings with students from different classes.

Track by seat number

When you record attendance, each attendees’ 5th-6th character (seat number) will be recorded, and generate a list of absentees according to missing seat numbers.

Best for meetings with students from the same class.
